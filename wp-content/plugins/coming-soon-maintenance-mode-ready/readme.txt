== Description ==

The [Coming Soon / Maintenance mode](http://readyshoppingcart.com/coming-soon-plugin/ "WordPress Coming Soon Plugin") Ready! plugin is the best solution for your website, while it's under construction.

Professional and simple way to setup launch page, the Coming Soon / Maintenance mode Ready! works with any theme on any hosting.

More over you can use it as:

* Coming Soon Page (pre-set templates and custom styles available)
* Maintenance Mode / Redirect 301
* UnderConstruction HTTP 503 - Service Unavailable

**Features**

* Works with any WordPress Theme
* Build with HTML5 & CSS3
* Responsive. Work on any device
* Access controls (Restrict By User Role, IP Address,)
* Social integration (Facebook, Tweeter, Google+)
* Multisite Support
* BuddyPress Support
* Free Coming soon page templates!
* Easy and user-friendly setup
* 3 Coming soon modes: Coming soon page, Redirect and show 503 HTTP
* Subscribe feature
* Flexible and easy coming soon page template setup
* SEO (Search Engine Optimization) options

== Installation ==

1. Download Coming Soon / Maintaince mode Ready! plugin
2. Upload to your WordPress plugins directory
3. Activate and enjoy
